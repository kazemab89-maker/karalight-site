
KaraLight — بسته آماده آپلود روی هاست (نسخه استاتیک)
تاریخ ساخت: 2025-08-11

محتوا:
- index.html (صفحه اصلی)
- catalog.html (کاتالوگ محصولات)
- about.html (درباره ما)
- faq.html (سؤالات متداول)
- contact.html (تماس با ما)
- assets/css/styles.css (استایل‌ها)
- assets/js/main.js (اسکریپت‌ها)
- assets/images/* (آیکون/پلیس‌هولدر)

نحوه استقرار:
1) کل پوشه karalight_site را به صورت ZIP آپلود کنید.
2) در File Manager هاست، فایل ZIP را Extract کنید تا index.html در ریشه دامنه قرار بگیرد.
3) اگر از ساب‌فولدر استفاده می‌کنید، اطمینان حاصل کنید مسیرها ثابت بماند (نسبی هستند).
4) برای فرم تماس، این نسخه استاتیک است. برای ارسال ایمیل واقعی، یک endpoint بک‌اند اضافه کنید.

یادداشت سئو:
- عبارات کلیدی: "تولید لاینر در مشهد"، "قیمت لاینر در مشهد"، "عکس لاینر مشهد"، "لاینر کناف"، "لاینر آویز".
- عنوان‌ها و توضیحات متا در هر صفحه تنظیم شده‌اند.
